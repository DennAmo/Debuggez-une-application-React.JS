toBe(value)
Vérifie si la valeur reçue est strictement égale à la valeur attendue (utilise ===).

toEqual(value)
Vérifie si les valeurs reçues et attendues sont équivalentes. Utilisé pour comparer des objets et des tableaux.

toBeNull()
Vérifie si la valeur reçue est null.

toBeUndefined()
Vérifie si la valeur reçue est undefined.

toBeDefined()
Vérifie si la valeur reçue n’est pas undefined.

toBeTruthy()
Vérifie si la valeur reçue est considérée comme true en contexte booléen.

toBeFalsy()
Vérifie si la valeur reçue est considérée comme false en contexte booléen.

toContain(item)
Vérifie si un tableau ou une chaîne de caractères contient un élément spécifique.

toHaveLength(length)
Vérifie si un tableau ou une chaîne a une longueur spécifique.

toMatch(regexp | string)
Vérifie si une chaîne correspond à une expression régulière ou une sous-chaîne.

toThrow()
Vérifie si une fonction lance une exception.

toHaveProperty(keyPath, value)
Vérifie si un objet a une propriété spécifique avec une valeur optionnelle.

equal(value)
Vérifie si la valeur reçue est strictement égale à la valeur attendue.

deep.equal(value)
Vérifie si les objets ou tableaux sont profondément égaux.

null
Vérifie si la valeur reçue est null.

undefined

true

false

include(item)
Vérifie si un tableau ou une chaîne de caractères contient un élément spécifique.

lengthOf(length)
Vérifie si un tableau ou une chaîne a une longueur spécifique.

match(regexp | string)
Vérifie si une chaîne correspond à une expression régulière ou une sous-chaîne.

throw(errMsg)
